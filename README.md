# Ticket Managment Website

Able employes to create ticket and check his tickets status


## Technologies used
* SQL
* C#
* Azure
* cshtml
* CSS
* Bootstrap 
* LINQ
* SQL Server


## User Stories
Employee must be able to:
* Sign up
* Log in 
* View his tickets
* create tickets

Admin must be able to:
* Sign up
* Log in 
* Show all tickets information
* Manage tickets by reject or approve it
